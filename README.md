# 🎴 Git Profile Card

> A simple and elegant web application to search and display GitHub user profiles with their repository information.

## ✨ Features

- 🔍 **Search GitHub Users** - Search for any GitHub user by username
- 👤 **User Profile Display** - View detailed user information including:
  - Profile picture
  - Name and username
  - Bio/Description
  - Followers count
  - Following count
  - Public repositories count
- 📚 **Recent Repositories** - Display user's latest repositories with direct links
- 🎨 **Clean UI** - Beautiful and responsive card-based design
- ⚡ **Real-time API** - Fetches live data from GitHub API
- 🚨 **Error Handling** - User-friendly error messages for invalid searches

## 🛠️ Technologies Used

- **JavaScript (ES6+)** - Core functionality with async/await
- **HTML5** - Semantic markup
- **CSS3** - Responsive styling
- **Axios** - HTTP client for API requests
- **GitHub API** - User and repository data
- **Font Awesome** - Icon library

## 📱 Responsive Design

The application is fully responsive and works seamlessly on:
- 💻 Desktop browsers
- 📱 Tablets
- 📲 Mobile devices
